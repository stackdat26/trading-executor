Build a complete automated paper trading executor with 
these exact specifications:

=== WHAT IT DOES ===
1. Connects to my existing trading bot's signals
2. Automatically places paper trades on Alpaca
3. Monitors open positions until TP or SL is hit
4. Logs every trade result to Google Sheets
5. Shows a simple dashboard of open positions and results

=== ALPACA INTEGRATION ===
Base URL: https://paper-api.alpaca.markets
API Key: loaded from environment variable ALPACA_KEY
Secret Key: loaded from environment variable ALPACA_SECRET

Order logic:
- BUY signal → place market buy order
- SELL signal → place market sell order
- Immediately after entry, place bracket order with:
  Stop loss: entry - (ATR × 2.0)
  Take profit: entry + (ATR × 3.0)
- Position size: $1000 per trade (paper money)
- Only trade US stocks and indices via Alpaca

=== SIGNAL SOURCE ===
Read signals from a JSON file called signals.json
that the trading bot writes to.
Format of each signal:
{
  "symbol": "NVDA",
  "action": "BUY",
  "entry": 164.71,
  "stop_loss": 154.52,
  "take_profit": 170.71,
  "confidence": 80,
  "daily_atr": 5.20,
  "timestamp": "2026-04-01 19:31:00"
}

=== POSITION MONITORING ===
Check all open positions every 5 minutes
For each position check if:
- Current price >= take profit → close position, log WIN
- Current price <= stop loss → close position, log LOSS
- Position open > 48 hours → close position, log EXPIRED

=== GOOGLE SHEETS LOGGING ===
Use Google Sheets API to log every trade
Columns:
Date | Time | Symbol | Direction | Confidence | 
Entry | Stop | Target | Exit Price | Outcome | 
PnL % | PnL $ | Duration | Notes

Credentials loaded from environment variable 
GOOGLE_SHEETS_CREDS (JSON string)
Sheet name: QuantBot Trade Log

=== DASHBOARD ===
Simple Flask web app on port 5000
Dark theme matching my other projects (#0d1117)
Shows:
- Account value (from Alpaca paper account)
- Total trades, wins, losses, win rate
- Open positions table
- Closed trades history table
- Equity curve (simple line chart)
Auto refreshes every 30 seconds
